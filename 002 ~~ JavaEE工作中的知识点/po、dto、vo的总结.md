
# 1. 命名规范（驼峰）

|   含义描述   |   命名规范示例   |
|:-----|:-----|
|   post body请求参数   |   XxRequest   |
|   展示层对象命名        |   XxVo   |
|   数据传输对象命名     |   XxDto   |
|   es实体名命名           |   XxIndexDO   |
|   db实体命名              |   跟表名相同   |
|   mongo实体命名        |   XxDoc   |
|   db组合关系实体名称  |   Xx   |
|   service接口命名        |   XxService   |
|   service实现命名        |   XxServiceImpl   |
|   manager,service引入多个manager进行负责的组合业务处理   |   XxManager   |
|   dao层命名                |   XxMapper   |
|   封装持久化组合服务（一个实体需要从db,es,redis多种存储获取）   |   XxRepository |
